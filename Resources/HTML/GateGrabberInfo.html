Gate Grabber Information

This puzzle game is based on logic gate functionality. In this game the player will choose an input (0 or 1) that will satisfy the gate that is on the screen. Each gate will have 2 inputs and 1 output. One of the inputs is given and the other comes from the player. There is the possibility for gates that do not have a correct answer. In that case, no input should be given to the gate.

When the game starts, there are 3 gates that fall at a chosen rate. Each gate will reset when the player provides an input or the gate falls off the screen. At this time, a new gate and rate is randomly generated.

If the player provides the correct value to satisfy the gate, the player receives 100 points. If the player provides an incorrect value to the gate, the player will lose a life. The player starts off with 10 lives and 0 points. The game ends when the player has 0 lives left. If the player gets a score in the top 10 they will be asked for their name for the highscore board.

There are powerups that the player can retrieve. There is a x2 multiplier and a 1up powerup right now. The x2 multiplier lasts for 15 seconds. The 1up gives the player 1 extra life.
