Gate Grabber is a game that is based on the concepts of basic Boolean logic gates. The goal of the game is to collect as many of the gates as you can by inputting the value that causes them to evaluate to a logical 1, or "true". When a gate is collected and the input evaluates to a 1 you will recieve points. However, collecting a gate that evaluates to a logical 0, or "false" will cause you to lose a life. There is no penalty for not collecting all the gates, only for inputting incorrect answers.
Hint: Not all gates have a possible correct answer!
